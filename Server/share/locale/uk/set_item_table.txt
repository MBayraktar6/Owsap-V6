# Set Item Table...
#
# There are two (2) types of sets, one equipping normal equipment and the other
# a set of costumes and/or additional items such as amulets or pets.
#
# In the example shown below, if the player is equipped with two (2) normal equipment of
# items of the same set value (6), the player will receive the following three (3) bonuses.
# However, if the player who is equipped with three (3) items instead, the bonuses
# in the example are the same but with higher values.
#
# Group DragonSetBonus
# {
# 	SET_VALUE	6
#
# 	1	2	MAX_HP	2500
# 	2	2	MALL_EXPBONUS	20
# 	3	2	ATTBONUS_MONSTER	15
# 	4	3	MAX_HP	3500
# 	5	3	MALL_EXPBONUS	30
# 	6	3	ATTBONUS_MONSTER	20
# }
#
# As for other types of sets, like costumes, in the example shown below, if the player is
# equipped with a body costume between the range `41682` ~ `41685` and a costume hair between
# the range `45360` ~ `45363`, the player will receive one (1) bonus because he has at least
# two (2) equipped items, according to the requirement. The same logic applies to more equipped
# items from the requirements below. The more equipped items, the higher and/or more bonuses the
# player will receive.
#
# Group DragonCostumeSetBonus
# {
# 	SET_VALUE	101
#
# 	SET_ITEM_COSTUME_BODY	41001	41685	1
# 	SET_ITEM_COSTUME_HAIR	45360	45363	1
#
# 	1	2	ATTBONUS_MONSTER	3
# 	2	3	ATTBONUS_MONSTER	3
# 	3	3	MALL_EXPBONUS	100
# 	4	4	ATTBONUS_MONSTER	5
# 	5	4	MALL_EXPBONUS	150
# }
#
# The strings below are all the current available set item tokens.
# Each token must be set with a minimum, maximum and a range value, if there is no maximum value
# then set the value to 0 or the minimum value.
#
# 	SET_ITEM_COSTUME_BODY		Body Costume
# 	SET_ITEM_COSTUME_HAIR		Hair Costume
# 	SET_ITEM_COSTUME_MOUNT		Mount Costume
# 	SET_ITEM_COSTUME_ACCE		Sash Costume
# 	SET_ITEM_COSTUME_WEAPON		Weapon Costume
# 	SET_ITEM_UNIQUE				Unique Item
# 	SET_ITEM_PET				Pet Item
#
# Important !!!
# The group structure must be respected, the value of the set must be at the
# beginning of the group defined as `SET_VALUE` along with its value separated by a tab.
# Below the `SET_VALUE` is where the equipment requirements and bonuses are set.
#
# When creating other sets, like costume sets, the `SET_VALUE` must be equal or
# above value `100` which marks the beginning of this type of set requirement.
#
# 2023.10.09.Owsap
# Owsap Development
#
